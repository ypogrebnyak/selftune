Installation

To install this template, just drop the files on your server. Make sure the to rename the file you want to be the frontpage to index.html.

Contact form

To get the contact form working, you need to have PHP installed on your server. Also, you have to choose an email address where the emails should be sent.

  1. Open up mailer.php, and change the email on line 4.
  2. Add the ID #contact_form to the form you want to use.
